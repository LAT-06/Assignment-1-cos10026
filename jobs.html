Fullstack Developer

Reference Number: FSD01

Job Description:

Design, develop, and maintain both front-end and back-end systems to deliver scalable, secure, and user-friendly solutions.

Salary Range:

AUD 85,000 – AUD 120,000 annually

Key Responsibilities

  1. Develop web applications using modern frameworks and technologies.
  2. scalable server-side APIs and database solutions.
  3. Collaborate with UI/UX designers for user-friendly interfaces.
  4. Optimize code for performance and scalability.

Requirements

  • Essential:

    Proficiency in HTML, CSS, JavaScript, React, Node.js, and database technologies.

  • Preferable:

    Experience with cloud platforms and CI/CD pipelines.

Successful Applicant

Database Manager

Reference Number: DB456

Job Description:

Responsible for designing, implementing, and maintaining the company’s database systems to ensure efficient and secure storage and retrieval of data.

Salary Range:

AUD 95,000 – AUD 135,000 annually

Key Responsibilities

  1. Design and implement database architectures to support the website.
  2. Manage database performance, backups, and recovery.
  3. Ensure data security and compliance with regulations.
  4. Stay updated on VR trends and technologies to propose innovative solutions.
  5. Collaborate with developers to integrate databases into the website.

Requirements

  • Essential:

    Bachelor’s in CS or related, 3+ years in database administration, expertise in MySQL/PostgreSQL/MongoDB, strong SQL and optimization skills, familiarity with data security standards.

  • Preferable:

    Experience with cloud databases (AWS/Azure), knowledge of Python or PL/SQL for database programming.

Successful Applicant

Reports to: Finance Lead Manager

IT Project Manager

Reference Number: PM789

Job Description:

Oversees the planning, execution, and delivery of the website development project, ensuring it meets objectives, deadlines, and budget constraints.

Salary Range:

AUD 100,000 – AUD 120,000 annually

Key Responsibilities

  1. Develop detailed project plans and schedules for website development.
  2. Coordinate team activities and ensure collaboration between stakeholders.
  3. Monitor project progress and resolve issues.
  4. Manage project budgets and resources effectively.
  5. Prepare and present project updates to senior management.

Requirements

Successful Applicant

UI/UX Designer

Reference Number: UXD01

Job Description:

Responsible for designing user-friendly and visually appealing website interfaces to enhance user experience and engagement.

Salary Range:

AUD 80,000 – AUD 115,000 annually

Key Responsibilities

  1. Design intuitive user interfaces and experiences for the website.
  2. Create wireframes, prototypes, and high-fidelity mockups.
  3. Collaborate with developers to ensure designs are accurately implemented.
  4. Conduct user research and usability testing to refine designs.
  5. Ensure the website is accessible and responsive across devices.

Requirements

  • Essential:

    Bachelor’s in Graphic/UI/UX Design, 2+ years in UI/UX design, proficiency in Figma/Adobe XD/Sketch, strong user-centered design knowledge, responsive design and accessibility expertise.

  • Preferable:

    Front-end development experience (HTML, CSS, JavaScript), familiarity with Agile methodologies.

Successful Applicant

Reports to: IT Infrastructure Manager
