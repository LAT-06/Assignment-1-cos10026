Overall Enhancements

Interactivity

We integrated AOS for smooth scrolling, interactive hover effects, card-flip animations, and glowing elements, making the UI more dynamic and engaging.

Responsiveness

We used CSS Flexbox, Media Queries, and viewport-based units to ensure adaptability across different screen sizes, enhancing usability and accessibility.

Specific Enhancements

Home Page

  • Navigation Bar: Used Flexbox to align and hover the menu items.
  • "Learn More" Button: Added CSS hover effects to change color and create a click effect.
  • Animations & Effects: Used AOS (Animate On Scroll) to create a smooth scroll effect.
  • Responsive Design: Used CSS Media Queries to optimize the interface on different devices.
  • Typography & Colors : Applied fonts from Google Fonts and used CSS Variables to manage colors.
  • Button Effects: Used box-shadow & transition to create click effects and change color on hover.
  • References:
    1. AOS Library (Scroll Animation)
    2. Google Fonts
    3. CSS Flexbox Guide
    4. CSS Button Effects

About Page

  • Interactive Slideshow: The group member images and details are displayed interactively by hovering over circular buttons.
  • Beyond Basics:
